Atalhos do VS Code

Atalhos gerais

Vamos ver alguns atalhos legais do VS Code que vão deixar a nossa vida mais alegre, supimpa e feliz.

ctrl+espaço: chama o autocompletar de um contexto.

alt+z: Quebra automática de linha

alt+setas cima/baixo: Move a selação para cima/baixo no arquivo. Se nada estiver selecionado, movimenta a linha em que o cursor está.

shift+alt+setas cima/baixo: Copia a selação para cima/baixo no arquivo. Se nada estiver selecionado, copia a linha em que o cursor está.

ctrl+; : comentário/descomentário

ctrl+shift+k: deletar linha
